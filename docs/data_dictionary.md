# Dictionnaire de Données

- age : int, âge du client en années.
- sexe : catégorie, sexe du client : H (Homme) ou F (Femme).
- situation_familiale : catégorie, statut marital (célibataire, marié, divorcé).
- revenu_mensuel : int, revenu mensuel du client en FCFA.
- type_contrat_travail : catégorie, type de contrat : CDI, CDD, Informel, Indépendant.
- anciennete_travail_mois : int, ancienneté au travail en mois.
- montant_credit : int, montant du crédit demandé en FCFA.
- duree_mois : int, durée du crédit en mois.
- taux_interet : float, taux d’intérêt annuel (%) du crédit.
- type_credit : catégorie, type de crédit : immobilier, auto, conso, scolaire…
- nb_retards_30j : int, nombre de retards sur 30 jours.
- nb_retards_60j : int, nombre de retards sur 60 jours.
- nb_retards_90j : int, nombre de retards sur 90 jours.
- taux_endettement : float, %, taux total d’endettement du client (%).
- ratio_dette_revenu : float, ratio dette sur revenu (montant mensuel crédit / revenu).
- defaut : int, variable cible (0 = pas de defaut (rembourse), 1 = défaut(ne rembourse pas)).
